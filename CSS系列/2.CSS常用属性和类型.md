# CSS常用属性和类型

### 一、常用属性

#### 1.字体属性

- 字体：font-family

- 字号：font-size

- 粗细：font-weight

- 斜体：font-style



#### 2.文本属性

- 颜色：color

- 对齐：text-align

- 装饰（下划线）：text-decoration

- 缩进：text-indent

- 行内元素间距：line-height

- 行内块元素对齐方式：vertical-align



#### 3.背景属性

- 背景颜色：background-color

- 背景图片：background-image

- 背景图片平铺：background-repeat

- 背景图片位置：background-position

- 背景图片固定：background-attachment
