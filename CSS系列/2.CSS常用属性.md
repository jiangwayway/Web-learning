# CSS常用属性

### 一、字体属性

字体：font-family

字号：font-size

粗细：font-weight

斜体：font-style



### 二、文本属性

颜色：color

对齐：text-align

装饰（下划线）：text-decoration

缩进：text-indent

行间距：line-height



### 三、背景属性

背景颜色：background-color

背景图片：background-image

背景图片平铺：background-repeat

背景图片位置：background-position

背景图片固定：background-attachment
